# Road Accident Blackspot Analysis and Visualization System

## Project Overview

This project analyzes historical road accident data to identify accident-prone locations (blackspots) and visualize them through an interactive dashboard.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Streamlit
* Matplotlib
* Seaborn

## Features

* Data Cleaning and Preprocessing
* Feature Engineering
* Blackspot Identification
* Data Visualization
* Interactive Dashboard
